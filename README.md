# Traveler's Kitchen

Generate simple, realistic recipes based on your available equipment. Perfect for hostels, guesthouses, and basic rental kitchens.

**Try it free:** [travelkitchen.app](https://www.travelkitchen.app)

## What it does

- Tell us what equipment you have (hob, microwave, kettle, etc.)
- Add any dietary requirements
- Get a complete recipe with shopping list and step-by-step instructions

## Status

Currently in beta. Free to use.

## Tech

- Next.js
- Convex
- Better Auth
- Tailwind CSS
